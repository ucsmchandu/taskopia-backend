# Chat and Communication in Taskopia

## Overview

Taskopia provides an in-platform chat system that allows Hosts and Allies to communicate safely about a task.

The chat feature is designed to help users coordinate task-related information while maintaining privacy and reducing spam or unwanted communication.

Chat is intended only for discussions related to an accepted task.

---

# When Does Chat Become Available?

Chat becomes available only after a Host accepts an Ally's application.

Before acceptance:

- No messages can be exchanged.
- Hosts cannot receive direct messages from applicants.
- Allies cannot contact Hosts privately.

This rule applies to every task on the platform.

---

# Why is Chat Restricted?

Taskopia limits communication before acceptance for several reasons.

These include:

- Preventing spam messages.
- Reducing harassment.
- Protecting user privacy.
- Preventing unnecessary distractions.
- Ensuring conversations remain task-related.
- Helping Hosts manage applications more efficiently.

This creates a fair and secure experience for everyone.

---

# Purpose of Chat

The chat system should be used only for information related to the accepted task.

Examples include:

- Confirming the meeting location.
- Discussing arrival time.
- Clarifying task requirements.
- Asking questions about the work.
- Informing about delays.
- Confirming task completion.

---

# Appropriate Communication

Users should communicate respectfully and professionally.

Good communication includes:

- Being polite.
- Responding promptly.
- Asking relevant questions.
- Providing accurate information.
- Keeping discussions focused on the task.

Respectful communication builds trust between Hosts and Allies.

---

# Inappropriate Communication

The chat system must not be used for:

- Harassment
- Hate speech
- Threats
- Spam
- Advertising
- Scams
- Fraud
- Offensive language
- Sharing illegal content
- Bullying
- Unwanted personal conversations

Such behavior violates Taskopia's community guidelines.

---

# AI-Powered Translation

Taskopia includes AI-powered translation to help users communicate across different languages.

Users may write messages in their preferred language.

The platform can translate messages so that both users understand the conversation more easily.

This helps:

- Reduce language barriers.
- Improve communication.
- Support multilingual communities.
- Increase accessibility.

---

# Message Privacy

Messages exchanged through Taskopia are intended only for the Host and the accepted Ally involved in the task.

Other applicants cannot view the conversation.

Taskopia respects user privacy while maintaining platform security.

---

# Message Moderation

Taskopia may review chat messages when necessary.

This helps investigate:

- Abuse reports
- Fraud
- Harassment
- Policy violations
- Safety concerns

Routine conversations are not manually reviewed unless required for moderation or support.

---

# Sharing Personal Information

Users should avoid sharing unnecessary personal information.

Examples include:

- Passwords
- Banking credentials
- Government identification numbers
- Sensitive financial information

Only share information that is necessary to complete the task.

---

# Reporting Chat Abuse

If a user experiences inappropriate communication, they should report the conversation through the Report Problem feature.

Examples of reportable behavior include:

- Threats
- Harassment
- Scams
- Fake payment requests
- Offensive language
- Spam
- Repeated unwanted messages

Taskopia reviews reports and may take action against accounts that violate platform rules.

---

# Communication Best Practices

Hosts should:

- Respond clearly.
- Explain expectations.
- Confirm meeting details.
- Inform Allies about changes.

Allies should:

- Reply promptly.
- Ask questions when unsure.
- Inform the Host if delayed.
- Confirm task completion.

Both users should remain respectful throughout the conversation.

---

# Safety Tips

For safer communication:

- Keep conversations inside Taskopia.
- Do not share passwords.
- Avoid sending confidential documents.
- Verify important information before meeting.
- Report suspicious behavior immediately.

---

# Frequently Asked Questions

## Can I chat before my application is accepted?

No.

Chat becomes available only after the Host accepts your application.

---

## Why can't I message the Host before acceptance?

This helps reduce spam, protects user privacy, and ensures that communication happens only between users who have agreed to work together.

---

## Can I send messages in my own language?

Yes.

Taskopia supports AI-powered translation to make multilingual communication easier.

---

## Can other applicants see my chat?

No.

Only the Host and the selected Ally can access the conversation.

---

## Can Taskopia read my messages?

Taskopia may review messages only when investigating abuse reports, safety concerns, or violations of platform policies.

---

## Can I report abusive messages?

Yes.

Users can report inappropriate conversations through the Report Problem feature.

---

## Should I share my phone number?

Only if it is necessary for completing the task.

Avoid sharing unnecessary personal information.

---

# Summary

Taskopia's chat system is designed to provide secure, respectful, and task-focused communication between Hosts and accepted Allies.

By restricting chat until acceptance, supporting AI-powered translation, encouraging professional behavior, and providing reporting tools, Taskopia helps create a safer and more trustworthy environment for everyone.
