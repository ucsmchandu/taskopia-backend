# Platform Features

## Overview

Taskopia provides a complete platform for discovering, posting, managing, and completing short-term tasks.

The platform is designed to simplify the entire task management process while ensuring safety, transparency, and ease of use.

Features are available based on the user's role, either as a Host or an Ally.

---

# User Authentication

Taskopia provides a secure authentication system.

Users can:

- Register a new account
- Log in securely
- Log out
- Reset forgotten passwords
- Verify their email address (if enabled)

Separate account types are available for:

- Hosts
- Allies

Authentication protects user accounts and platform resources.

---

# User Profiles

Every registered user has a personal profile.

A profile may include:

- Name
- Profile picture
- Bio
- Skills
- Contact information (where applicable)
- Ratings
- Reviews
- Completed tasks
- Account role

Keeping profile information updated improves trust and visibility.

---

# Host Dashboard

The Host dashboard allows Hosts to manage all their activities from one place.

Hosts can:

- View posted tasks
- Create new tasks
- Edit tasks
- Delete tasks
- Review applications
- Accept an Ally
- View completed tasks
- Track task status

The dashboard provides an organized overview of all posted work.

---

# Ally Dashboard

The Ally dashboard helps Allies manage their work opportunities.

Allies can:

- Browse tasks
- View applied tasks
- Track application status
- View accepted tasks
- View completed tasks
- Access chat
- Monitor ratings and reviews

The dashboard serves as the Ally's personal workspace.

---

# Task Posting

Hosts can create detailed task listings.

Each task includes:

- Title
- Description
- Category
- Budget
- Date
- Time
- Duration
- Location
- Additional instructions

Well-written tasks receive better applications.

---

# Task Discovery

Allies can browse available tasks.

Taskopia allows users to discover work based on:

- Category
- Location
- Keywords
- Budget
- Availability

This makes finding suitable work fast and convenient.

---

# Search and Filters

Taskopia includes search functionality to help users find relevant tasks.

Users can filter tasks using:

- Category
- Budget
- Location
- Date
- Keywords

Search helps reduce the time required to discover opportunities.

---

# Task Applications

Interested Allies can apply for available tasks.

Hosts receive applications for review.

For every task:

- Multiple applications are allowed.
- Only one Ally can be selected.

Application status is updated as the task progresses.

---

# Application Management

Hosts can:

- View applicants
- Compare applicants
- Review profiles
- Accept one Ally

The selection process remains entirely under the Host's control.

---

# In-Platform Chat

Taskopia includes secure messaging between Hosts and accepted Allies.

Chat becomes available only after acceptance.

Users can discuss:

- Meeting location
- Timing
- Task requirements
- Clarifications

The chat system supports task-related communication.

---

# AI Task Description Generator

Taskopia includes AI assistance during task creation.

Hosts can generate professional task descriptions using AI.

Benefits include:

- Better clarity
- Faster task creation
- Improved readability
- Reduced writing effort

---

# AI Translation

Taskopia supports multilingual communication.

AI Translation allows users to:

- Translate task descriptions
- Translate chat messages

This helps users communicate even if they speak different languages.

---

# AI Help Assistant

Taskopia includes an AI-powered Help Assistant.

The chatbot answers platform-related questions using the Taskopia knowledge base.

The assistant can explain:

- Features
- Policies
- Workflows
- User roles
- Safety guidelines

The assistant is powered by Retrieval-Augmented Generation (RAG).
