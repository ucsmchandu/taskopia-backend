# Allies in Taskopia

## Who is an Ally?

An Ally is a registered Taskopia user who applies for and completes tasks posted by Hosts.

Allies use Taskopia to discover flexible, short-term work opportunities that match their skills, interests, availability, and location.

Unlike traditional employees, Allies are free to decide which tasks they want to apply for and when they want to work.

Taskopia provides a platform where Allies can earn income while helping individuals, businesses, and local communities complete temporary work.

---

# Purpose of Allies

Allies are the workforce of the Taskopia community.

They help Hosts complete tasks efficiently while gaining experience, earning money, and building their reputation through completed work and positive reviews.

Taskopia allows Allies to work independently without long-term commitments.

---

# Who Can Become an Ally?

Anyone looking for flexible, temporary work can become an Ally.

Examples include:

- College students
- Freelancers
- Skilled workers
- Part-time workers
- Homemakers
- Retired professionals
- Local residents
- Job seekers
- Individuals wanting extra income

No specific profession is required, although some tasks may require particular skills or experience.

---

# Responsibilities of an Ally

Allies are expected to act professionally throughout the task lifecycle.

Their responsibilities include:

- Browsing available tasks
- Reading task details carefully
- Applying only for tasks they can complete
- Communicating respectfully
- Arriving on time
- Completing work as agreed
- Following the Host's reasonable instructions
- Maintaining professional behavior
- Leaving honest ratings and reviews

---

# Discovering Tasks

Taskopia allows Allies to browse tasks posted by Hosts.

Tasks can typically be explored based on:

- Category
- Location
- Budget
- Date
- Keywords

Reading task details carefully helps Allies decide whether the task matches their availability and skills.

---

# Understanding Task Details

Before applying, Allies should review all available information.

This may include:

- Task title
- Description
- Category
- Location
- Budget
- Date
- Time
- Estimated duration
- Required skills
- Special instructions

Understanding these details reduces misunderstandings later.

---

# Applying for Tasks

If an Ally is interested in a task, they can submit an application.

An application expresses the Ally's willingness to complete the task.

Applying does not guarantee selection.

Hosts review all applications before choosing one Ally.

---

# Multiple Applications

Allies may apply for multiple tasks simultaneously.

Each Host independently reviews their own applicants.

Receiving one acceptance does not automatically cancel other applications unless platform rules specify otherwise.

---

# Waiting for Acceptance

After submitting an application, the Ally waits for the Host's decision.

Possible outcomes include:

- Accepted
- Not selected
- Task cancelled

Hosts choose applicants based on their own preferences and requirements.

Taskopia does not automatically select applicants.

---

# Acceptance

When an Ally is selected:

- The application is accepted.
- In-app chat becomes available.
- Task preparation can begin.
- Both users coordinate final details.

Only one Ally can be accepted for each task.

---

# Chat After Acceptance

The chat system becomes available only after acceptance.

The chat should be used for task-related communication, including:

- Meeting location
- Arrival time
- Task clarification
- Necessary preparations

Professional communication helps ensure successful task completion.

---

# Preparing for the Task

Before the scheduled time, Allies should:

- Review the task details again.
- Confirm the location.
- Arrive on time.
- Carry required tools if necessary.
- Inform the Host if unexpected delays occur.

Preparation increases trust and improves the overall experience.

---

# Completing the Task

During task execution, Allies should:

- Follow agreed instructions.
- Complete the work responsibly.
- Respect the Host's property.
- Behave professionally.
- Inform the Host when the work is finished.

Task completion takes place offline.

Taskopia does not supervise offline interactions.

---

# Receiving Payment

Payment is handled according to the agreement between the Host and the Ally.

The agreed budget should be understood before the task begins.

Taskopia acts as a platform connecting users and does not automatically manage payments unless additional payment features are introduced.

---

# Ratings and Reviews

After task completion, Hosts may rate and review the Ally.

Positive ratings help Allies:

- Build credibility
- Improve visibility
- Increase trust
- Receive more opportunities

Honest reviews contribute to a reliable community.

---

# Building a Strong Profile

A strong Ally profile increases the chances of being selected.

Helpful practices include:

- Completing tasks successfully
- Receiving positive ratings
- Maintaining professional communication
- Applying only to suitable tasks
- Being punctual
- Keeping profile information updated

---

# Best Practices for Allies

Successful Allies generally:

- Read task descriptions carefully.
- Apply only when available.
- Respond promptly.
- Communicate respectfully.
- Arrive on time.
- Complete tasks responsibly.
- Maintain professionalism.
- Leave honest reviews.

These habits improve reputation and increase future opportunities.

---

# What Allies Should Avoid

Allies should never:

- Submit fake applications.
- Apply without intending to work.
- Harass Hosts.
- Request unnecessary personal information.
- Misrepresent their skills.
- Fail to appear without notice.
- Perform illegal activities.
- Use Taskopia for fraudulent purposes.

Violations may result in warnings, suspension, or permanent account termination.
