# Task Categories in Taskopia

## Overview

Taskopia supports a wide range of short-term tasks across different industries and everyday situations.

The platform is designed for temporary work rather than permanent employment. Tasks may require physical work, technical skills, creative abilities, or general assistance.

Choosing the correct category helps Allies discover relevant tasks more easily and improves search accuracy.

---

# Home Services

Home Services include tasks related to maintaining, improving, or assisting within residential properties.

These tasks are commonly posted by homeowners, tenants, or families.

Examples include:

- Deep cleaning
- Kitchen cleaning
- Bathroom cleaning

- Lawn mowing
- Furniture assembly
- House shifting
- Appliance installation
- Wall painting
- Minor home repairs
- Plumbing assistance
- Electrical assistance
- Window cleaning

Suitable Hosts:

- Homeowners
- Families
- Apartment residents

# Choosing the Right Category

Hosts should always choose the category that best represents the work.

Choosing the correct category helps:

- Improve task visibility
- Reach relevant Allies
- Increase application quality
- Improve search results
- Reduce confusion

---

# Category Guidelines

When creating a task:

- Select the closest matching category.
- Write a detailed description.
- Mention any required skills.
- Specify the location.
- Clearly state the payment.
- Mention expected duration.

If a task could belong to multiple categories, choose the one that best describes the primary work.
